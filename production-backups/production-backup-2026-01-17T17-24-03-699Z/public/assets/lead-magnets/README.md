# Lead Magnet Assets

This directory contains the lead magnet files that users can download after subscribing to the newsletter.

## Required Files

To complete the lead magnet funnel, you need to add the following files:

### Excel Files (.xlsx)
- `ai-tools-roi-calculator.xlsx` - Main ROI calculator for AI tools
- `business-ai-roi-calculator.xlsx` - Business-focused ROI calculator

### PDF Files (.pdf)
- `ai-agents-automation-guide.pdf` - AI agents automation guide
- `ai-coding-tools-comparison.pdf` - Coding tools comparison
- `creative-ai-workflow-guide.pdf` - Creative workflow guide
- `ecommerce-ai-strategy.pdf` - E-commerce AI strategy
- `marketing-ai-tools-guide.pdf` - Marketing AI tools guide

## File Requirements

1. **File Size**: Keep files under 10MB for fast downloads
2. **Format**: Use PDF for guides, Excel for calculators
3. **Branding**: Include AI Buzz Media branding
4. **Content**: Ensure high-quality, valuable content
